## 继承线问题

### 演算逻辑：


```
L(O)  := [O]                                                

L(A)  := [A] + merge(L(O), [O])           
       = [A] + merge([O], [O])
       = [A, O] 
                                    
L(B)  := [B, O]                    
L(C)  := [C, O]

L(K1) := [K1] + merge(L(A), L(B), [A, B])          
       = [K1, A] + merge([O], [B, O], [B])       
       = [K1, A, B] + merge([O], [O])          
       = [K1, A, B, O]

L(K2) := [K2, A, C, O]  

L(Z)  := [Z] + merge(L(K1), L(K2), [K1, K2])
       = [Z] + merge([K1, A, B, O], [K2, B, C, O], [K1, K2])   
       = [Z, K1] + merge([A, B, O], [K2, B, C, O], [K2])      
       = [Z, K1, K2] + merge([A, B, O], [B, C, O])   
       = [Z, K1, K2, C] + merge([A, B, O], [B, O])
       = [Z, K1, K2, C, A] + merge([B, O], [O])
       = [Z, K1, K2, C, A, B] + merge([O], [O])
       = [Z, K1, K2, C, A, B, O]
       
```
